# Ribari Extension Examples

Use it for two things:

- install example extensions by copying their folders into `~/.config/ribari/extensions/`
- study the included docs and SponsorBlock example when porting a Chrome-style extension to Ribari

Contents:

- `docs/extensions.md` — extension API, install guide, and porting reference
- `sponsor-block/` — complete tile-plus-content-script example

Install an example extension:

1. Quit Ribari.
2. Copy the example folder into `~/.config/ribari/extensions/`.
3. Relaunch Ribari and open the extension from the Apps sidebar or command palette.
